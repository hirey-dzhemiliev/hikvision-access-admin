# Обзор системы

## Что это за система

Это локальное веб-приложение на Python и SQLite для учета сотрудников и синхронизации данных с панелями Hikvision `DS-KV9503-WBE1`.

Система рассчитана на небольшой объект:

- `2-3` панели;
- около `30` сотрудников;
- без сложной иерархии ролей и без большого количества временных посетителей.

Главная идея: локальная база является источником истины, а панели получают актуальные данные из неё.

## Что уже умеет текущая версия

- авторизация админки из `YAML`;
- карточки сотрудников с фото, картой, комнатой и статусом;
- список панелей и проверка доступности;
- настройка времени и аутентификации по лицу;
- предпросмотр и применение синхронизации;
- история запусков синхронизации;
- экран расхождений;
- журнал проходов со снимками;
- массовые действия по сотрудникам;
- карточный и табличный режим сотрудников;
- подтверждения для опасных операций.

## Основные разделы интерфейса

- главная панель;
- сотрудники;
- панели;
- синхронизация;
- история синхронизаций;
- расхождения;
- журнал проходов;
- настройки.

## Минимальные сущности

### Сотрудник

- `employee_id`
- `first_name`
- `last_name`
- `room_number`
- `card_number`
- `photo_path`
- `lifecycle_state`
- `comment`

### Панель

- `name`
- `host`
- `username`
- `password`
- `enabled`
- `time_zone`
- `time_display_format`
- `time_mode`
- `manual_time`
- `ntp_server`
- `ntp_port`
- `ntp_interval`
- `face_auth_enabled`
