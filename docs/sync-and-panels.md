# Панели и синхронизация

## Подтвержденные настройки панели

Для `DS-KV9503-WBE1` в приложении подтверждены и встроены:

- часовой пояс;
- формат отображения даты и времени;
- переключение `manual / NTP`;
- ручная установка точного времени;
- настройка `NTP` сервера;
- включение и выключение аутентификации по лицу.

Настройки доступны:

- в карточке отдельной панели;
- на странице `Панели` как массовая форма для применения к выбранным панелям.

## Принцип синхронизации

Синхронизация односторонняя:

- `SQLite -> панели Hikvision`

Перед применением всегда строится diff:

- кто будет создан;
- кто будет обновлён;
- кто есть только на панели;
- кто будет удалён с панели.

Удаления происходят только после явного подтверждения.

## Что важно сейчас

- после запуска синхронизации приложение повторно считает preview;
- интерфейс показывает уже актуальное состояние, а не старый diff;
- обычные страницы не должны делать live-запросы к панели по каждому блоку;
- при открытии страниц админки обновляется кэш доступности панелей и `sync preview`.

## Fallback-логика для Hikvision

В коде уже есть обработка типичных особенностей прошивок:

- обязательный блок `Valid` при создании пользователя;
- удаление через `UserInfo/SetUp`, если `UserInfoDetail/Delete` не поддерживается;
- корректная обработка `deviceUserAlreadyExistFace`;
- более подробные ошибки после Digest auth.
